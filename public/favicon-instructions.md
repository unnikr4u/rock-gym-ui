# Favicon Setup Instructions

## Files needed in this directory:

1. **favicon.ico** - Main favicon file (16x16, 32x32, 48x48 pixels)
2. **logo192.png** - Apple touch icon and PWA icon (192x192 pixels)  
3. **logo512.png** - Large PWA icon (512x512 pixels)

## How to create these files:

1. Save your Rock Fitness Centre logo image
2. Go to https://favicon.io/favicon-converter/
3. Upload your logo image
4. Download the generated favicon package
5. Copy the following files to this directory:
   - favicon.ico
   - android-chrome-192x192.png (rename to logo192.png)
   - android-chrome-512x512.png (rename to logo512.png)
